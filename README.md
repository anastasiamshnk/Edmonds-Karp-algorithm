# Edmonds-Karp-algorithm
алгоритм Эдмондса-Карпа
Maximum flow: 23

Этот код решает задачу о максимальном потоке в сети, где у нас есть ориентированный граф с весами на ребрах, а также две вершины: исток и сток. Задача заключается в том, чтобы найти такой поток максимальной величины, который может протекать из истока в сток, учитывая ограничения на пропускную способность ребер. Алгоритм Эдмондса-Карпа является улучшением алгоритма Форда-Фалкерсона и использует поиск в ширину для нахождения кратчайшего увеличивающего пути в остаточной сети.
